# Starter Guide — `/starter` book

Page-by-page copy for the virtual book. Kept short on purpose: a Minecraft
written book fits ~14 lines / ~256 characters per page, so each task is its own
page (or two). `[FILL IN]` marks server-specific values.

---

**Page 1 — Welcome**
```
Welcome to Theatria!

This book is your
guide. Do the tasks
in order — they only
take a few minutes.

Reopen it anytime:
   /starter

Turn the page  >>
```

**Page 2 — Task 1: Find your spot**
```
TASK 1
Find your spot

Type  /rtp  to drop
into the wild.

Don't like it?
/rtp again until you
find a place you
love.
```

**Page 3 — Task 2: Set your home**
```
TASK 2
Set your home

Found your spot?
Lock it in:

   /sethome

Now  /home  brings
you back anytime.
```

**Page 4 — Task 3: Make your first money**
```
TASK 3
First money

Grab gear:
   /kit starter

Mine stone or chop
wood. Hold an item
and check its value:
   /worth
```

**Page 5 — Task 3 (cont.): Sell it**
```
Sell what's in your
hand:
   /sell hand

GOAL: earn $[FILL IN]
to claim your land.

Fastest & safe:
[FILL IN method]
```

**Page 6 — Task 4: Claim your land**
```
TASK 4
Claim your land

Stand where you want
to build, then:
   /claim

This protects your
stuff from other
players.
```

**Page 7 — Task 5: Rank up**
```
TASK 5
Rank up

Keep selling to grow
your balance, then:
   /rank up

Higher rank =
more perks.
```

**Page 8 — Bonus + help**
```
BONUS
```
