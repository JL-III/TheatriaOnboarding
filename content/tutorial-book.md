# Tutorial Guide — `/tutorial` book

Page-by-page copy for the virtual book. Kept short on purpose: a Minecraft
written book fits ~14 lines / ~256 characters per page, so each task is its own
page (or two). `[FILL IN]` marks server-specific values.

---

**Page 1 — Welcome**
```
Welcome to Theatria!

This book is your
guide. Do the tasks
in order — they only
take a few minutes.

Reopen it anytime:
   /tutorial

Turn the page  >>
```

**Page 2 — Task 1: Set out to find a spot**
```
TASK 1
Set out to find
a spot

Walk through the
spawn portal to
reach the wild.

Not satisfied? Use
/rtp until you find
a spot you like,
then continue your
/tutorial tasks.
```

**Page 3 — Task 2: Set your home**
```
TASK 2
Set your home

Found your spot?
Lock it in:

   /sethome

Now  /home  brings
you back anytime.
```

**Page 4 — Task 3: Make your first money**
```
TASK 3
First money

Grab gear:
   /kit welcome

Mine stone or chop
wood. Hold an item
and check its value:
   /worth
```

**Page 5 — Task 3 (cont.): Sell it**
```
Sell what's in your
hand:
   /sell hand

GOAL: $1,000 - the
cost of your first
claim.

Mine & sell
cobblestone, coal,
copper.
```

**Page 6 — Task 4: Claim your land**
```
TASK 4
Claim your land

Stand where you want
to build:
   /lands create
   <name>

Costs $1,000 and
creates your land.
Protects it from
other players.
```

**Page 7 — Task 5: Rank up**
```
TASK 5
Rank up

Keep selling to grow
your balance, then:
   /rank up

Higher rank =
more perks.
```

**Page 8 — Bonus + help**
```
BONUS

Play ~{min} min to grab
your DAILY REWARD —
easy money to start.

Stuck? Just ask in
chat. Have fun!
```

> `{min}` is filled in live by the renderer from TheatriaSessions' real reward
> threshold (active, non-AFK minutes), so the book never promises a number that
> drifts from config. When TheatriaSessions is absent it shows `daily-minutes`.
> Note: Sessions counts **active** time — AFK minutes don't count toward it.
